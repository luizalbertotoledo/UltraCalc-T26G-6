# UltraCalc - Repositório Template

Este repositório é o modelo oficial do projeto **UltraCalc**, utilizado como base para novas turmas e novas execuções do projeto.

Ele contém apenas a estrutura inicial do sistema, organizada para desenvolvimento colaborativo utilizando Git e GitHub.

## 📁 Estrutura do Projeto
```
UltraCalc/
│
├── principal.py # Arquivo principal que inicia o sistema
├── interface.py # Telas e mensagens exibidas ao usuário
├── operacoes.py # Funções matemáticas
├── historico.py # Manipulação do histórico
├── erros.py # Registro de falhas
│
├── dados/
│ ├── historico.txt
│ ├── erros_log.txt
│ └── empresa.txt
│
└── testes/
└── resultados_testes.txt
```

## 🎯 Objetivo do Projeto
Desenvolver uma calculadora modular chamada **UltraCalc**, aplicando:
- Organização de código em múltiplos arquivos
- Controle de versões com Git
- Trabalho colaborativo com branches e Pull Requests
- Code Review
- Gestão de tarefas com GitHub Projects

## 🔄 Fluxo de Trabalho Esperado
1. Criar uma branch para cada tarefa.
2. Desenvolver localmente.
3. Realizar commit com mensagem clara.
4. Enviar para o repositório remoto (`push`).
5. Criar Pull Request.
6. Passar por revisão.
7. Realizar merge após aprovação.
